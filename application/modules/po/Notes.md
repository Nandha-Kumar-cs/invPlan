To achieve the outlined processes within the `po` folder, follow these steps:

1. **Create the `tempdata/` Function**:
    - This function will allow data entry.
    - Store the entered data in a temporary storage (e.g., a database or file).

2. **View and Download CSV Data File**:
    - Provide a functionality to view the entered data.
    - Add an option to download the data as a CSV file.

3. **Open ATS File and Data File**:
    - Create a function that opens the ATS file and reads the data from the CSV file.

4. **Automatically Fetch Data from `data.csv`**:
    - Implement a function that automatically fetches data from the `data.csv` file and populates it in the required format.

5. **Upload the ATS XLSM File**:
    - Allow users to upload the ATS XLSM file.

6. **Sendmail Process**:
    - Create a process to send an email with the provided data to a recipient.
